# Ebook Prompter

Ebook Prompter is a React-based application designed to enhance your reading experience. It provides a smooth and customizable reading interface with features tailored for speed reading and user convenience.

## Features

1. **Smooth Scrolling**: Enjoy a seamless reading experience with optimized scrolling for large font sizes and high reading speeds.
2. **Dark Mode Toggle**: Easily switch between light and dark modes using a convenient checkbox toggle.
3. **Granular WPM Increments**: Adjust your reading speed with more precise Words Per Minute (WPM) increments.
4. **Horizontal Scrolling for WPM Presets**: Quickly navigate through WPM presets with a horizontal scrolling interface.
5. **Custom Favicon**: A unique favicon has been added for better branding and user experience.
6. **Reading State Save/Restore**: Automatically save your reading progress on pause, exit, or every minute, and restore it when you return.
7. **Chapter Scroll Reset**: When changing chapters, the scroll resets to the beginning of the new chapter.
8. **Play Scroll Behavior**: Resumes reading from the last reading point when the play button is pressed.
9. **Clickable Words**: Click on any word to change the reading point instantly.

## Technologies Used

- React
- Zustand for state management
- Screen Wake Lock API
- Local Storage
- CSS for styling
- HTML for favicon integration

## How to Use

1. Clone the repository.
2. Install dependencies using `npm install`.
3. Start the development server with `npm run dev`.
4. Open the application in your browser and enjoy your reading experience.
